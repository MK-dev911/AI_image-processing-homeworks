# 🖼️ AI Image Processing - Homework Repository

This repository contains my homework assignments for the **Artificial Intelligence: Image Processing** course. Each assignment focuses on different image processing concepts, computer vision techniques, and AI-based methods implemented throughout the semester.

## 📚 Course Topics

The assignments may cover topics such as:

- Image loading and preprocessing
- Color space transformations
- Image filtering and enhancement
- Histogram analysis and equalization
- Edge detection
- Image segmentation
- Feature extraction
- Morphological operations
- Object detection
- Pattern recognition
- Machine Learning for image analysis
- Deep Learning for computer vision

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/MK-dev911/AI_image-processing-homeworks.git
cd AI_image-processing-homeworks
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run a homework:

```bash
python Homework-01/solution.py
```

or open the notebook:

```bash
jupyter notebook
```
